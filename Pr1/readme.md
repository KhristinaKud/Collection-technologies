# Практична 1 
Об’єднати “замовлення” з довідником товарів (створити окремий DataFrame з категоріями/собівартістю), виконати merge, знайти рядки без відповідників і оцінити їхню частку.
## Пояснення програми 
За цим сайтом [click](https://realpython.com/pandas-merge-join-and-concat/), дізнаємось що таке marge і як воно працює.За допомогою допоміжних засобів створюємо Dataframe замовлень та товарів. Використовуємо Left marge для об’єднання таблиці замовлень із довідником товарів за спільним ключем product_id. Якщо опис товару відсутній у довіднику, у такому разі замість заповнених рядків  з'являються значення NaN.Далі розраховується частка помилкових замовлень і виводяться результати.
