# Практична 1 
Об’єднати “замовлення” з довідником товарів (створити окремий DataFrame з категоріями/собівартістю), виконати merge, знайти рядки без відповідників і оцінити їхню частку.
## Пояснення програми 
За цим сайтом [click](https://realpython.com/pandas-merge-join-and-concat/), дізнаємось що таке marge і як воно працює.За допомогою допміжних засобів створюємо Dataframe замовлень та товарів. Використовуємо Left marge для об’єднання таблиці замовлень із довідником товарів за спільним ключем product_id. Якщо воно не знаходить такий товар, то використовується фільтрація, де замість даних з'явилися значення NaN.Розраховується частка помилкових замовлень і виводяться результати.
